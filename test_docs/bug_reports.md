# Баг-репорты TaskFlow

## BUG-001: Отсутствие валидации due_date в прошлом

| Поле | Значение |
|------|----------|
| **ID** | BUG-001 |
| **Заголовок** | Система позволяет установить due_date в прошлом |
| **Окружение** | Backend API v1.0, любая ОС |
| **Шаги воспроизведения** | 1. Авторизоваться в системе<br>2. Создать задачу с due_date = "2020-01-01" (прошлая дата)<br>3. Проверить сохранённую задачу |
| **Ожидаемый результат** | Ошибка валидации или предупреждение |
| **Фактический результат** | Задача создаётся с прошедшей датой |
| **Severity** | Minor |
| **Priority** | Medium |
| **Статус** | Open |

---

## BUG-002: Изоляция данных возвращает 404 вместо 403

| Поле | Значение |
|------|----------|
| **ID** | BUG-002 |
| **Заголовок** | Доступ к чужой задаче возвращает 404 вместо 403 |
| **Окружение** | Backend API v1.0 |
| **Шаги воспроизведения** | 1. Создать пользователя А и задачу А<br>2. Создать пользователя Б, получить его токен<br>3. Отправить GET `/tasks/{id задачи А}` с токеном пользователя Б |
| **Ожидаемый результат** | HTTP 403 Forbidden (Нет прав доступа) |
| **Фактический результат** | HTTP 404 Not Found |
| **Severity** | Major |
| **Priority** | High |
| **Статус** | Open |

---

## BUG-003: Нет возможности очистить описание при обновлении

| Поле | Значение |
|------|----------|
| **ID** | BUG-003 |
| **Заголовок** | При обновлении задачи нельзя очистить описание |
| **Окружение** | Backend API v1.0 |
| **Шаги воспроизведения** | 1. Создать задачу с description = "Some text"<br>2. Обновить задачу с description = ""<br>3. Получить задачу |
| **Ожидаемый результат** | description = "" |
| **Фактический результат** | description остаётся "Some text" |
| **Severity** | Minor |
| **Priority** | Low |
| **Статус** | Open |

---

## BUG-004: Некорректный расчёт completion_rate при отсутствии задач

| Поле | Значение |
|------|----------|
| **ID** | BUG-004 |
| **Заголовок** | При пустом списке задач completion_rate возвращает NaN |
| **Окружение** | Backend API v1.0 |
| **Шаги воспроизведения** | 1. У пользователя нет задач<br>2. Запросить GET `/statistics/tasks-summary` |
| **Ожидаемый результат** | completion_rate = 0 |
| **Фактический результат** | completion_rate = NaN (деление на ноль) |
| **Severity** | Minor |
| **Priority** | Medium |
| **Статус** | Fixed |

## BUG-005: Округление avg_priority до 2 знаков

| Поле | Значение |
|------|----------|
| **ID** | BUG-005 |
| **Заголовок** | avg_priority возвращается без двух знаков после запятой |
| **Окружение** | Backend API v1.0 |
| **Шаги воспроизведения** | 1. Создать задачи с приоритетами 1, 2, 3<br>2. Запросить статистику |
| **Ожидаемый результат** | avg_priority = "2.00" |
| **Фактический результат** | avg_priority = 2.0 (не хватает нулей) |
| **Severity** | Trivial |
| **Priority** | Low |
| **Статус** | Open |
